## Getting Started

Clone the project

```bash
git clone https://github.com/chingu-voyages/V61-tier2-team-23
```

## Run the project locally

```bash
cd Guessify
```

Create .env and add your keys:
```bash
[ENV KEYS]
```

```bash
npm install
```

```bash
npm run dev
```

Open browser

```bash
http://localhost:5173/
```
